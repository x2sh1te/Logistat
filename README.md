# Logistat — Учет грузоперевозок (MVP)

Мобильное приложение для локального учета рейсов, заказов и финансов, разработанное специально для частных перевозчиков (водителей).

## Основные возможности
- **Два профиля водителей**: Быстрое переключение между профилями на стартовом экране.
- **Локальный учет**: Все данные хранятся на устройстве (SQLite/Room), интернет не требуется.
- **Управление заказами**: 
    - Просмотр списка рейсов за последний год.
    - Поиск по маршруту или заказчику.
    - Быстрое удаление и редактирование через свайпы (жесты).
    - Цветовая индикация статуса оплаты (Оплачено/Долг/Частично).
- **Умная форма ввода**:
    - Выбор даты через календарь.
    - Справочник заказчиков (автоматическое сохранение новых имен).
    - Кнопка копирования маршрута для быстрой отправки данных.
- **Аналитика и Экспорт**:
    - Сводка выручки (нал/безнал) за выбранный период.
    - Формирование отчета в формате CSV (Excel).
    - Возможность отправить отчет в Telegram, WhatsApp или по почте.

## Стек технологий
- **Язык**: Kotlin
- **UI**: Jetpack Compose (Modern Toolkit)
- **База данных**: Room (SQLite)
- **Архитектура**: MVVM (Model-View-ViewModel)
- **Навигация**: Jetpack Navigation

## Как запустить
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/x2sh1te/Logistat.git
   ```
2. Откройте проект в **Android Studio Ladybug (или новее)**.
3. Дождитесь окончания синхронизации Gradle.
4. Запустите приложение на эмуляторе или реальном устройстве через кнопку **Run**.
